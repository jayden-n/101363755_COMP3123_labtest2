> [!NOTE]

## Installation

1. Clone project in your terminal

```shell
git clone https://github.com/jayden-n/101363755_COMP3123_labtest2.git
```

2. Navigate to the project

```shell
cd 101363755_COMP3123_labtest2
```

3. Install dependencies

```shell
npm i
```
4. Starting Application

```shell
npm start
```
